# flutter_application_1

ビンゴアプリ

## 目標

- タイトル画面、ビンゴ画面、設定画面から構成される。
    - タイトル画面
        - タイトル表示、ビンゴ画面、設定画面への遷移のみ行う。
    - ビンゴ画面
        - ビンゴの実行、設定画面への遷移を行う。
        - ビンゴの制御はBLocパターンを用いて行う。
    - 設定画面
        - ビンゴに関する設定、ビンゴ画面への遷移を行う。
        - 
    - その他
        - 演出にアニメーションを用いる。
        - テストコードを作成する。
